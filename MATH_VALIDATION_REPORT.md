# MATH VALIDATION REPORT — L.U.M.U.S.-8 PURE Edition

## 1. 原式の評価

### Efficiency = Composite Return / Volatility

リスク調整済みモメンタムとして直感的ですが、分母推定誤差に弱く、さらに逆ボラ配分まで行うと低ボラを二重優遇します。改善版では `Efficiency` を診断列として残し、ランキング本体は **Momentum 55% / Quality 25% / Low Volatility 20%** と分離しました。

### Quality = 平均上昇日リターン / 平均下落日リターン

上下の日数を無視し、下落日が少ない・小標本の銘柄で不安定です。改善版は、252日間の上昇日比率と対数価格トレンドの決定係数を各50%とした、0〜1の経路品質指標に置換しました。なお、これは企業の財務Qualityではありません。

### Valuation_Alt = 高値からの距離 × 逆ボラティリティ

バリュエーションではありません。原式 `1 / proximity_to_high / vol` は52週高値から遠く、低ボラな銘柄を高評価し、モメンタムと矛盾します。改善版では廃止しました。真の割安度を導入するなら、時点整合した財務データが必要です。

### 40:40:20 と通常z-score

不安定なQualityへ40%を配分し、外れ値に弱い通常z-scoreを使うため、単一異常値が順位を支配します。改善版は中央値/MADによるrobust z-scoreを±3でclipします。重みは理論上の唯一解ではなく、四半期リバランス・税引後のウォークフォワード検証対象です。

## 2. 改善版モメンタム

評価日の直近21営業日を除外し、次を合成します。

- 12-1モメンタム: 50%
- 6-1モメンタム: 30%
- 3-1モメンタム: 20%

短期反転への過剰反応を抑えつつ、構造変化への反応性を残します。最低274観測を要求し、不足銘柄を代替始点で計算しません。正の合成モメンタムかつ200日線上のみ適格です。

## 3. 配分

各市場上位6銘柄という固定枠は地域分散を保証しますが、地域間の機会差を無視します。PURE版のマンデートとして維持し、6銘柄未満を黙って他地域で補完せず警告します。逆ボラ配分には1銘柄 **4%〜12%** の制約を設定し、極端な集中を防ぎます。

## 4. レジーム

200日線は単純・説明可能で、遅行性と往復売買の弱点があります。全銘柄スコアへの一律減算は順位を変えないため無意味です。改善版では銘柄適格性に200日線を使用し、市場指数レジームによる100%/60%/20%は上位プロトコルが承認する「助言稼働率」として分離しました。

## 5. 未検証事項

A評価の最終承認前に、point-in-timeユニバースで税・手数料・スリッページ・売買回転率を含むウォークフォワード検証が必要です。重みや閾値を同じ期間で最適化・評価してはいけません。
